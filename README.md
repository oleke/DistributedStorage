## Folder Structure
The workspace contains two folders by default, where:

- `src`: the folder to maintain sources

## Compilation & Running
# To compile: 
- `cd src`
- `javac *.java -d ../bin`

# To run: 
- `cd bin`
- To run the master Node: `java MasterNode`
- To run a data node: `java DataNode`
- To run a client: `java Client`
